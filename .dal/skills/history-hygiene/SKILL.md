# History Hygiene

## Rules
1. **최종 결과만 기록.** 중간 시도, 실패한 접근은 기록하지 않는다.
2. **확인 전에는 기록하지 않는다.** 추측, 미완성 결과 금지.
3. **결정이 뒤집히면 즉시 갱신.** 과거 기록을 삭제하거나 취소 표시.
4. **12KB 초과 금지.** scribe가 Core Context로 압축.

## Format
```
### {날짜}: {요약}
**상태:** 완료 | 진행 중 | 중단
**결과:** {핵심 결과}
**다음:** {후속 액션}
**주의:** {다음 세션이 알아야 할 것}
```

## Anti-Pattern: Kobayashi
중간 상태를 history에 남기면 다음 세션 dal이 잘못된 정보를 읽는다.
"되는 것 같다"는 기록하지 않는다. "됐다" 또는 "안 됐다"만 기록한다.
